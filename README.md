# Vigil Adoption Snapshot

Static frontend demo for Vigil adoption messaging.

## Pages

- `index.html` - interactive Slack-style Vigil product demo (frontend only)
- `adoption-slide.html` - animated adoption metrics + onboarding CTA slide

## Notes

- This repository is intentionally frontend-only.
- No backend or Slack integration is included.
- Intended for video/demo capture and internal storytelling.
